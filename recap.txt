create the djongo project : django-admin startproject projectName

create a subject app : python manage.py startapp appName

create a urls.py in my app

in the appName.urls.py import
	from . import views
	from django.urls import path
	
in the appName.views 
	from django.http import HttpResponse

now in projectName.urls
	from django import include
	path("", include("appName.urls"))

now to run the local server : python manage.py runserver


create a templates folder in main(wherever your manage.py is there)
	add that templates folder in settings.py DIR = [BASE_DIR, "templates"]

then in the viewa.py use(instead of HttpResponse) : render(request, "name.html")

dynamic or static data 
in views.py make a dict and add all the variables then name of the dict in render : render(request, "name.html", dictName)
In html you have to use "{{varName}}" to access it 

How use GET method : it used with the name of the field: varName = request.GET["feildName"]

How to use POST method : Same as get but you have to add method = 'POST' in form and add {% csrf_token %} 

CSRF(Cross site rquest forgery): CSRF is a security vulnerability where an attacker tricks a user's browser into making an unauthorized request to a trusted site where the user is authenticated. Django provides built-in protection against CSRF attacks.
CSRF token: Unique secret unpredictable value

HOW TO CSS FILES TO HTML FILES:
create a static named file where you created the templates file
and then do to settings and import os
navigate to static and type: STATICFILES_DIRS = (os.path.join(BASE_DIR, 'static'),)
then go to your html file and type: {% load static %} and then <link rel="stylesheet" href="{% static 'styles.css' %">
